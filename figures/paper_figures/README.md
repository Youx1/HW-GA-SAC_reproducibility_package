# Revision Figure Set

The PDF files in this folder include the manuscript figures using the same filenames as the submitted LaTeX source:

Main-text figures:
- `fig03_seven_method_outcomes.pdf`: seven-method success, collision, and timeout outcomes.
- `fig04_all_method_5seed_stability.pdf`: five-seed success, collision, and timeout outcomes for all seven methods.
- `fig05_all_method_path_stretch.pdf`: successful-path stretch comparison for the reported methods.
- `fig06_city_split_success.pdf`: training-city replay versus unseen-layout success split.
- `fig07_safety_ablation_success_200ep.pdf`: 200-episode separated HW_GA_SAC actor-side safety-input ablation.
- `fig08_sensor_stress_success.pdf`: synthetic sensor noise and observation-latency stress check.

Appendix figures:
- `figA1_success_return.pdf`: seven-method success-return relationship.
- `figA2_city_layout_topdown.pdf`: city42 and city1-4 top-down layout montage.
- `figA3_sensor_noise_latency_grid.pdf`: full sensor noise, latency, and combined perturbation grid.
- `figA4_extended_unseen_city_generalization.pdf`: city43-52 extended generated-layout evaluation.
- `figA5_wind_robustness.pdf`: no-wind versus strong-wind success under the simplified force model.

Additional PNG/SVG exports and earlier alias filenames are retained only for convenience and backward compatibility with the plotting scripts.
